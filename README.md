# Micro-Frontends

## module-genarator

To build a module

```
yarn build:modules --module=<module name>
```

e.g.

```
yarn build:moudles --module=TodoList
yarn build:moudles --module=AddTodo
```

To build vendor file

```
yarn build:vendors
```

To start static server on port 8002

```
yarn start
```
